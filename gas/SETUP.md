# 先生用セットアップ：生徒の回答をスプレッドシートに集める

生徒が「先生に送信」を押すと、**Googleスプレッドシートに1行ずつ回答が追記**され、先生が一覧で確認できるようになります。先生が**最初に一度だけ**この設定をします（10分ほど）。

## 手順

### 1. スプレッドシートを作る
1. Googleドライブ（学校アカウント）で **新しいスプレッドシート**を作成（名前は「首里まちあるき 回答」など）。

### 2. Apps Script を貼り付ける
2. そのスプレッドシートの上メニュー **拡張機能 → Apps Script** を開く。
3. 最初の `function myFunction() {}` を**すべて消して**、このフォルダの **`Code.gs` の中身を全部コピペ**して保存（💾）。

### 3. ウェブアプリとしてデプロイ
4. 右上の **デプロイ → 新しいデプロイ**。
5. 種類（⚙️歯車）で **ウェブアプリ** を選ぶ。
6. 設定：
   - **次のユーザーとして実行**：自分（あなた）
   - **アクセスできるユーザー**：**全員**
7. **デプロイ** を押す → 初回は権限の確認（自分のアカウントで許可）。
8. 表示される **ウェブアプリのURL**（`https://script.google.com/macros/s/……/exec` で終わるもの）を**コピー**。

### 4. アプリにURLを設定
9. アプリのフォルダにある **`config.js`** を開き、`submitUrl` にコピーしたURLを貼る：
   ```js
   window.MACHI_CONFIG = {
     submitUrl: 'https://script.google.com/macros/s/XXXXXXXX/exec'
   };
   ```
10. 保存して、サイトを更新（再公開）すれば完了。生徒の「先生に送信」がこのシートに届きます。

> ※ Claude に「config.js にこのURLを設定して」と頼めば、貼り付け＆再公開まで代わりにやります。

## 動作チェック
- ブラウザでウェブアプリURL（`/exec`）を開いて「回答受付：OK」と出ればデプロイ成功。
- 生徒画面で名前を入れて「📤 先生に送信」→ スプレッドシートの「回答」シートに行が増えればOK。

## 集まる項目（1行＝1人の送信）
送信時刻／名前／学年／班・チーム／コース／到達（◯/◯）／クイズ（◯/◯）／キーワード／スポット詳細／メモ／元データ(JSON)

## よくある質問
- **写真は届く？** いまは届きません（写真は各端末内に保存。届くのは「写真の枚数」と到着時刻・メモ・クイズ結果）。写真も集めたい場合は別途対応可能（Googleドライブ保存）。
- **あとからコードを直したら？** Apps Script で **デプロイ → デプロイを管理 → 編集（鉛筆）→ バージョン:新しいバージョン → デプロイ** で更新。URLは変わりません。
- **いたずら送信が心配**：URLを知っている人なら送信できます。気になる場合は合言葉（トークン）方式に変更できます（要相談）。
